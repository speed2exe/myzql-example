# myzql-example
Usage examples for myzql

# This example uses the following version
- myzql: 0.16.0
- zig: 0.16.0

## Setup and adding dependency to your project
- Look into `build.zig.zon` and `build.zig`

## Usage
- Look into `src/main.zig`

## Run the examples

### Prerequisite
- MySQL instance running(modifiable)
  - port: 3306
  - host: localhost
  - user: root
  - password: password

### Run
```bash
zig build run
```
